Layout

The Grid

The grid is built around three key elements: containers, rows, and columns. Containers create base padding for the page; rows create a max-width and contain the columns; and columns create the final structure.

Based on the 12 column Bootstrap grid. Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns. Including predefined classes for easy layout options. Echo Calendar layout uses single set of .col-sm-* grid classes, you can create a basic grid system that starts out stacked on mobile devices and tablet devices (the extra small to small range) before becoming horizontal on desktop (medium) devices. Place grid columns in any .row.

In the Grid you can nest columns down as far as you'd like. Just embed rows inside columns and go from there. Each embedded row can contain up to 12 columns <div class="row"> <div class=".col-sm-8"> <div class=".col-sm-4"> </div>

The layout for the Echo Marketing Calendar is optimized to respond from High Definition Desktop monitors down to tablet devices (such as iPads & MS Surfaces) only. The Grid is wrapped in a fluid container class class="container-fluid" with a minimum width set to min-device-width : 768px

Colors — is almost the most important part of the Echo Brand. The Echo Brand Colour Scheme is based on a complimentary colour rule based on a set of design principles that specifically relate to color by using two opposing colors on the color wheel to create a theme with complementary colors.

For your convenience we also provide Swatches Preset (Echo-Complementary-ColorSwatches.ase in the Pack folder).

Typography

Fonts

The Echo Online Brand uses two san-serif fonts from the Google Font API. Headers use Montserrat and body text such as paragraphs call the Open Sans font.

@import url(https://fonts.googleapis.com/css?family=Open+Sans:400,800,700,300);
@import url(https://fonts.googleapis.com/css?family=Montserrat:700,400);

Less Variables
@font-family-sans-serif: 'Open Sans', "Helvetica Neue", Helvetica, Arial, sans-serif;
@headings-font-family: 'Montserrat', sans-serif;

Buttons

When to use

Buttons are tied to an action of some kind (i.e. Submitting a form or closing a modal window).

When to use Anchor Tags

When you need the button to take you somewhere (e.g. a sign up button), this is the perfect solution.

Less Variables

.btn-default { .button-variant(@btn-default-color; @btn-default-bg; @btn-default-border); }
.btn-primary { .button-variant(@btn-primary-color; @btn-primary-bg; @btn-primary-border); }
